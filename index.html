// features

Everything a small editor should do, done properly.

Five languages, one editor

Python, C, C++, C#, and Batch all get real syntax highlighting. Open a file and the language is detected automatically — or pick one yourself from the Language menu.

A console you can talk to

Run a file with F5 and type straight into the output, right where the prompt is. set /p name=, scanf, input() — they all just work.

A caret that glides

The insertion point eases between positions instead of snapping, and scrolling animates too. Small detail, makes everything feel calmer.

Six themes, zero fuss

Dark+, Dracula, Monokai, Nord, Solarized Light, and Pure Light — switch instantly from the View menu or the theme picker above.

Shortcuts that respect your hands

Duplicate, delete, and reorder lines without leaving the keyboard. Toggle comments per-language. Find, replace, and jump to any line.

Tabs, saved settings, no fuss

Work across multiple files at once. Your theme, font, and preferences are remembered the next time you open the editor.

// console

Programs that ask questions get answers.

Most editors give you a read-only output box, which falls apart the moment a script calls set /p, scanf, or input(). IVP Editor's console is genuinely interactive — the text you type lands right after the prompt, exactly like a real terminal.
